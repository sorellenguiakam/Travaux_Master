## ⚡ Démarrage rapide

```bash
# 1. Créer l'environnement
conda create -n dt1_cameroun python=3.9
conda activate dt1_cameroun

# 2. Installer les dépendances
pip install -r requirements_full.txt

# 3. Lancer Jupyter
jupyter notebook

# 4. Ouvrir le premier notebook
# 3_NOTEBOOKS_PEDAGOGIQUES/Semaine01_Introduction_Python_ML.ipynb
```
