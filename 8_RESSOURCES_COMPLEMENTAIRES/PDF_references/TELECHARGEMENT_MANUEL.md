## 📝 INSTRUCTIONS GÉNÉRALES

### Comment télécharger depuis Frontiers:

1. **Ouvrir l'article** dans le navigateur (lien fourni ci-dessus)
2. **Attendre que la page charge complètement**
3. **Chercher le bouton "Download PDF"** (généralement en haut à droite)
4. Si le bouton n'apparaît pas:
   - Désactiver les bloqueurs de publicité
   - Rafraîchir la page
   - Essayer un autre navigateur (Chrome, Firefox)

### Comment télécharger depuis arXiv:

1. **Ouvrir le lien** de l'article
2. Cliquer sur **"PDF"** dans le menu de droite
3. Le PDF s'ouvre dans le navigateur
4. **Enregistrer sous** avec le nom spécifié

### Si un article n'est pas accessible:

1. **Vérifier le DOI** dans `liens_bibliographie.md`
2. **Essayer via Sci-Hub** (dernier recours, zone grise légale):
   - https://sci-hub.se/
   - Coller le DOI de l'article
3. **Contacter la bibliothèque universitaire** pour accès institutionnel
4. **Contacter l'auteur directement** via ResearchGate ou email
