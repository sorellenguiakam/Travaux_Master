## 📥 Comment Télécharger les Articles

### 1. Articles Open Access (gratuits)

**Frontiers, PLOS, BMC, arXiv** → Téléchargement direct

**Exemple:**
- Katte et al. (2023): https://www.frontiersin.org/articles/10.3389/fpubh.2023.1110013/full
- Cliquer sur "Download PDF"

### 2. Articles Payants (via institution)

**The Lancet, Nature, NEJM** → Accès via université/bibliothèque

**Options:**
1. **Accès institutionnel:** Se connecter avec identifiants université
2. **Bibliothèque universitaire:** Demander l'article via le service de prêt inter-bibliothèques
3. **ResearchGate:** Contacter directement l'auteur pour une copie

### 3. Livres Gratuits

- **Elements of Statistical Learning:** https://web.stanford.edu/~hastie/ElemStatLearn/
- **Interpretable ML:** https://christophm.github.io/interpretable-ml-book/
- **Murphy PML:** https://probml.github.io/pml-book/book1.html

- Identification rapide de l'année
- Cohérence pour les citations

## 🗂️ Gestion Bibliographique

### Outils recommandés:

1. **Zotero** (RECOMMANDÉ)
   - Gratuit, open-source
   - Extension navigateur pour import automatique
   - Export vers BibTeX pour LaTeX
   - Lien: https://www.zotero.org/

2. **Mendeley**
   - Gratuit (version de base)
   - Synchronisation cloud
   - Lien: https://www.mendeley.com/

3. **JabRef** (BibTeX natif)
   - Gratuit, spécialisé LaTeX
   - Lien: https://www.jabref.org/

### Workflow recommandé:

1. **Télécharger** le PDF depuis la source
2. **Renommer** selon convention (Auteur_Année_Titre.pdf)
3. **Classer** dans sous-dossier approprié
4. **Importer** dans Zotero/Mendeley
5. **Extraire** métadonnées (DOI, abstract, keywords)
6. **Exporter** vers bibliographie.bib (pour LaTeX)

## ⚠️ Vérification de la Qualité des Sources

### Critères de sélection:

✅ **Excellentes sources:**
- Journals impact factor > 5 (Nature, Science, Lancet, NEJM, JAMA)
- Conférences top tier (NIPS, ICML, KDD, CVPR)
- Livres publiés par éditeurs reconnus (Springer, MIT Press)

✅ **Bonnes sources:**
- Journals impact factor 2-5 (PLOS ONE, Frontiers, BMC)
- Conférences spécialisées reconnues
- Thèses de doctorat d'universités reconnues

⚠️ **Sources à valider:**
- Preprints arXiv (non peer-reviewed, mais OK si récent et pertinent)
- Working papers
- Blogs techniques (uniquement si auteur reconnu)

❌ **Sources à éviter:**
- Sites web non académiques
- Wikipédia (OK pour démarrer, mais citer les sources primaires)
- Articles sans DOI ni auteur identifiable
