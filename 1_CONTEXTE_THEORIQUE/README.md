## 📄 Contenu

### fiches_concepts_ml.md
Fiches explicatives sur les concepts clés du projet:

**Machine Learning:**
- Types d'apprentissage (supervisé, non supervisé)
- Classification binaire
- Overfitting vs Underfitting
- Biais-Variance Trade-off

**Algorithmes:**
- Arbres de décision (Decision Trees)
- Random Forest (bagging)
- Gradient Boosting (XGBoost, LightGBM)
- Support Vector Machines (SVM)
- Régression Logistique

**Métriques:**
- Accuracy, Precision, Recall, F1-score
- Matrice de confusion
- Courbe ROC et AUC
- Precision-Recall curves

**Techniques avancées:**
- Validation croisée (K-Fold)
- Optimisation hyperparamètres (GridSearch, RandomSearch)
- Gestion déséquilibre (SMOTE, class weighting)
- Interprétabilité (SHAP, LIME)

**Biomarqueurs:**
- ANP32A-IT1 (AUC > 0.8)
- ESCO2 (AUC > 0.8)
- NBPF1 (AUC > 0.8)
